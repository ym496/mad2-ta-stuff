The difference between "Now" and "Later"

🔄 Synchronous (Sync)

Execution happens line-by-line. JavaScript waits for each task to finish before moving to the next. It "blocks" the thread—if a function takes 10 seconds, the whole page freezes for 10 seconds.

⏳ Asynchronous (Async)

Execution is deferred. JS starts the task, moves it to the background, and continues running other code. When the task is ready, it joins a queue (Micro or Macro) to be finished later.

// 1. Sync (Runs Now)
console.log("Sync 1");

// 2. Async Macro (Runs Later)
setTimeout(() => console.log("Macro"), 0);

// 3. Async Micro (Runs "Sooner" Later)
Promise.resolve().then(() => {
  console.log("Micro");
});

console.log("Sync 2");

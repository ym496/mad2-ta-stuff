📦 Object Destructuring: The "Unpacker"

        

            
Stop repeating yourself! Instead of item.id, item.val, and item.type, just unpack them into independent variables.

            
// The FIRST item from your result above:
const item = { id: ?, val: ?, type: ? };

// 🪄 Destructuring Magic
const { id, val, type } = item;

console.log(type); // No 'item.' prefix needed!

📦 Object Destructuring: Renaming Variables

    

        
Sometimes you don't like the property name in the object. You can rename it while you unpack it!

        
// The Object:
const item = { id: 101, val: 5, type: 'Gear' };

// 🪄 Destructuring + Renaming:
// Read as: "Get 'val' but call it 'amount'"
const { id, val: amount, type } = item;

console.log(amount); // Result: 5
